# 💰 Daily Expense Splitter (Java Swing)

A simple Java Swing-based desktop application to split expenses among multiple people.

---

## 🚀 Features

- Enter number of people  
- Add names (comma-separated)  
- Enter amount paid by each person  
- Automatically calculates who owes whom  
- Simple and user-friendly UI  

---

## 🧠 Tech Stack

- Java  
- Swing (GUI)  

---

## ⚙️ How to Run

1. Compile:
```bash
javac ExpenseSplitterUI.java
